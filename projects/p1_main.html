Project One: Chicago Weather Data Scraping & Power BI Dashboard

This project showcases the use of Selenium and Beautiful Soup to automate the scraping of real-time weather data for the Chicago area. After scraping, the data is cleaned and uploaded into Power BI, where it is transformed into an intuitive, visually streamlined, and interactive dashboard, highlighting key weather trends and insights.

I am currently working on implementing automatic data refreshes to ensure the dashboard remains up to date without manual refreshes.
